# Chat Quality, Pacing, and Memory Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Eliminate the repetition / static-standoff failure mode and extend the GM's story memory across long campaigns by wiring four coordinated changes (context expansion, scene state tracker, prompt surgery, memory architecture) into a single backend slice.

**Architecture:** Two new Campaign columns (`last_scene_state` JSON, `narrative_summary` Text) persist GM-emitted structural state across turns. The `record_turn` tool schema gains a required `scene_state` object and optional `narrative_summary_update` string. The context builder injects scene_state diff, story-so-far, repeated-phrase banlist, and `gm_only` memories into the hidden brief while using `player_safe_summary` for older turn excerpts. System prompt gains a scene-change-discipline section, a seeded stock-phrase banlist, and three new pacing rules.

**Tech Stack:** Python 3.12, FastAPI, SQLAlchemy, Alembic, Anthropic + OpenAI SDKs, pytest. Spec: `docs/superpowers/specs/2026-05-12-chat-quality-pacing-design.md`.

**Files:**

- [ ] **Step 1: Add the helper**

In `backend/app/services/context_builder.py`, add this near the top of the file (after the imports, before `_RECENT_TURN_LIMIT`):

```python
import re
from collections import Counter

_PHRASE_NGRAM_SIZES = (5, 4)
_PHRASE_MIN_TURNS = 2
_PHRASE_MAX_RESULTS = 6
_WORD_RE = re.compile(r"[a-z0-9]+")


def _extract_repeated_phrases(recent_turns: list) -> list[str]:
    """Return up to 6 phrases (4- or 5-grams) that appear across 2+ distinct GM turns.

    Longer phrases beat shorter ones at the same frequency.
    """
    per_turn_phrases: list[set[str]] = []
    for turn in recent_turns:
        text = (getattr(turn, "gm_response", "") or "").lower()
        words = _WORD_RE.findall(text)
        seen: set[str] = set()
        for size in _PHRASE_NGRAM_SIZES:
            for i in range(0, len(words) - size + 1):
                seen.add(" ".join(words[i : i + size]))
        per_turn_phrases.append(seen)

    counts: Counter[str] = Counter()
    for phrases in per_turn_phrases:
        for phrase in phrases:
            counts[phrase] += 1

    ranked = [
        phrase
        for phrase, count in counts.items()
        if count >= _PHRASE_MIN_TURNS
    ]
    ranked.sort(key=lambda p: (-counts[p], -len(p)))

    deduped: list[str] = []
    for phrase in ranked:
        if any(phrase in existing or existing in phrase for existing in deduped):
            continue
        deduped.append(phrase)
        if len(deduped) >= _PHRASE_MAX_RESULTS:
            break
    return deduped
```
